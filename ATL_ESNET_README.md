## How to Create Remaining Pages

Each content page should follow this structure:

1. **Same header/navigation as existing pages**
2. **Breadcrumb navigation**: Home > Handbook/Blueprint > Page Title
3. **Page title and subtitle**
4. **Content sections** with:
   - Section titles (h2)
   - Subsection titles (h3) where needed
   - Content text in paragraphs
   - Highlight boxes for important information
   - Bullet lists where appropriate
5. **Back button** to return to main handbook/blueprint page
6. **Same footer as other pages**

## File Naming Convention
